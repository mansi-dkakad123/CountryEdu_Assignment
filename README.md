## 📌 Project Description

---

## 1. N-Queen Problem

This project implements the classical **N-Queens problem** using the **Backtracking algorithm**.

In this problem, we place **N queens on an N × N chessboard** such that no two queens can attack each other.

### Highlights

* Implementation of the backtracking algorithm.
* Safety check for each queen placement.
* Supports board sizes from **1 to 9**.
* Displays the total number of possible solutions.

---

## 2. Regular Expression Matcher

This program implements a regular expression matcher that checks whether a given string matches a specified pattern.

### Supported Feature

* `+` → Matches **one or more occurrences**

### Possible Outputs

* `True`
* `False`
* `Invalid Pattern`
